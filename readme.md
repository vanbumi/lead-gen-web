# LeadGen Pro

Extract leads from Google Maps + AI Outreach Messages.

## Features
- Scrape businesses from Google Maps
- Generate AI-powered outreach messages
- Export to Excel
- Scraping history

## Environment Variables
- `GROQ_API_KEY`: Your Groq API key for AI messages (set in Hugging Face Secrets, not in this file)

## Usage
1. Enter keyword (e.g., "cafe in bandung")
2. Select number of leads
3. Click Start Scraping
4. Download Excel file
